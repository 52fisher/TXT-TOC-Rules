# TXT-TOC-Rules

## 简介
为TXT小说分章规则提供常用目录的正则表达式。

目录规则已基本完善，重大更新时更新。


## 调试工具
项目提供 `debug-tool.html` 调试工具，用于：
- 可视化编辑和管理目录规则
- 拖拽排序规则
- 测试正则表达式匹配效果
- 支持导入TXT小说进行真实测试
- 导入/导出JSON规则文件

### 使用方法
1. 在浏览器中打开 `debug-tool.html`
2. 在左侧规则列表中编辑规则
3. 点击「调试」按钮在右侧查看测试结果
4. 支持切换「内置测试数据」或「导入的TXT小说」进行测试
5. 完成后点击「导出JSON」保存规则

## 规则说明
规则按 `serialNumber` 顺序执行，兜底规则(serialNumber=99)固定在最后。

### 主要规则类型
- 基础目录匹配
- 数字/中文数字标题
- 特殊符号标题
- Chapter/Section/Part英文标题
- 各种格式变体
